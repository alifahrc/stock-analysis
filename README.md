# stock-analysis
This stock analysis is based on one of my task for Investment and Portfolio Management. Stock and scenario analysis for AAPL, AMZN and GE from 2019 to 2024 which covers:
1. Understanding analysis for each stocks under different period and circumstances.
2. Understanding underlying economic condition and company decisions that might have impacted stocks' historical result.
3. Guidance for investors to pick stocks in their portfolio based on Sharpe Ratio.
4. Another important factor to consider in APT models.
5. Scenario analysis based on APT model factor.
